# Portfolio de Fulbert Ouattara

Ce site web est un portfolio personnel présentant mes compétences, projets et informations de contact.

## Fonctionnalités

- Sections : Home, About, Services, Portfolio, Contact
- Thème switcher avec plusieurs couleurs
- Formulaire de contact qui ouvre le client email avec le message pré-rempli

## Comment fonctionne le formulaire de contact

Le formulaire utilise une approche simple : lorsqu'un visiteur remplit et envoie le formulaire, il ouvre automatiquement leur client email (comme Outlook, Gmail, etc.) avec le message pré-rempli. L'utilisateur doit alors envoyer l'email manuellement depuis leur client.

Aucune configuration n'est requise - ça fonctionne immédiatement !

## Technologies utilisées

- HTML5
- CSS3
- JavaScript
- Typed.js pour l'effet de frappe

## Utilisation

Ouvrez `index.html` dans un navigateur web pour voir le site.

## Dépannage

- Si le client email ne s'ouvre pas, assurez-vous d'avoir un client email configuré sur votre ordinateur.
- Vérifiez la console du navigateur pour les logs de débogage.
